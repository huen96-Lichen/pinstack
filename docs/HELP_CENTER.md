# PinStack 帮助中心

## 1. PinStack 是什么

PinStack 是一个帮助你管理复制内容和截图内容的桌面工具。它会把你日常工作中的信息碎片自动保存下来，并在你需要时快速找到。

## 2. 它能帮你做什么

- 防止复制内容丢失
- 把文本和图片整理成可查找的资料
- 快速筛选提示词、修复方案、参考资料
- 直接复制复用，减少重复整理时间

## 3. 工作流程

PinStack 的核心流程是：

复制/截图 → 自动入库 → 分类管理 → 搜索筛选 → 再次使用

你不需要手动保存每一条内容，系统会先帮你收集，再帮你快速定位。

## 4. Dashboard 结构说明

### TitleBar（标题栏）

- 显示窗口标题与窗口级控制
- 包含固定置顶开关与帮助入口

### TopBar（操作栏）

- 用于搜索、筛选和系统控制
- 包含运行模式（Mode）、状态提示（Status）和弹出策略入口

### Sidebar（分类导航）

- 用于切换你“正在看什么”
- 包含一级导航（全部/文本/图片/收藏/AI 工作区）
- 在 AI 工作区下可进一步按用途查看

### Card List（记录列表）

- 展示具体记录内容
- 每条记录可进行复制、整理后复制、编辑、删除、再固定等操作

## 5. 关键概念说明

### 记录

一条被保存的内容，可以是文本或图片。

### 用途（Type）

记录的主要用途分类，例如：提示词、生成结果、问题修复、操作流程、参考资料、待整理。

### 标签

你自定义的辅助标记，用于跨分类组织内容。

### 来源

记录来自哪个应用或场景，帮助你按上下文定位内容。

### 模式（Mode）

系统运行方式，决定记录与弹出行为。

### 状态（Status）

系统当前可用性提示，例如是否存在权限问题。

## 6. 模式说明（Auto / Custom / Off）

### 自动模式（Auto）

- 图片自动弹出
- 文本自动弹出
- 适合需要积极捕获与提醒的场景

### 自定义模式（Custom）

- 由你决定图片/文本是否自动弹出
- 适合希望精细控制打扰程度的场景

### 关闭模式（Off）

- 停止弹出
- 停止记录
- 适合临时不希望系统介入的场景

## 7. 当前阶段特点（本地优先）

当前阶段以本地能力为主：

- 记录与管理流程优先在本机完成
- 重点是“可分类、可搜索、可复用”
- 不依赖复杂云端流程即可稳定使用
