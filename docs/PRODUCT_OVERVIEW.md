# PinStack 是什么

PinStack 是一款面向 AI 工作流的桌面效率工具。它会自动捕获你复制的文本和截图，把碎片信息变成可检索、可复用的工作资产。

## 一句话介绍

PinStack 是一个把“复制内容”变成“可复用 AI 工作资产”的工具。

## 核心价值（解决什么问题）

你在 ChatGPT、Codex、终端、文档之间反复切换时，最常见的痛点是：

- 复制过的内容很快就丢了
- 需要时找不到
- 找到了也不方便直接复用

PinStack 的价值是把“瞬时内容”沉淀成“长期资产”，让你从“临时保存”升级到“持续复用”。

## 适合谁

PinStack 适合以下用户：

- 经常在 AI 工具与项目之间来回工作的开发者
- 需要高频复制命令、提示词、修复步骤的技术团队
- 需要在多个任务中复用资料的内容创作者与知识工作者
- 希望建立个人素材库、减少重复劳动的重度电脑用户

## 核心能力

- 自动捕获：复制文本、截图后自动入库
- 结构化管理：支持用途分类（Type）、标签、来源
- 快速检索：通过导航、搜索和筛选快速定位内容
- 即时复用：支持复制、整理后复制、编辑、再固定
- 系统状态可视化：通过 Mode 与 Status 快速理解系统当前状态
- 本地优先：数据默认保存在本地，降低信息外泄风险

## 一句话总结

PinStack 让你不只是“存住内容”，而是把内容真正“用起来”。
