# PinStack v2.6.0 Release Note

## 定位
- 本版本为实验封版交付版本。
- 核心目标：让本地模型能力在设置页可见、可理解、可检查状态，并保持现有主流程稳定。

## 主要变更
- 新增设置页「本地模型（Local Model）」模块。
- 新增本地模型状态读取/刷新链路：`settings.localModel.status`。
- 新增模型选择入口：`settings.localModel.model.set`（当前仅支持 `gemma3:12b`）。
- 保持既有约束：
  - 唯一模型 `gemma3:12b`
  - `mock/real` 双模式
  - `summary` 并行旁路
  - `dedupeSuggestion` 建议模式
  - 自动重命名不覆盖用户手改标题

## 封版产物
- `dist/PinStack-2.6.0-arm64.dmg`
- `dist/PinStack-2.6.0-arm64.dmg.blockmap`

## 验证结论
- `npm run typecheck` 通过
- `npm run package` 通过
- 可生成可分发 DMG，未做签名/公证时可能在目标机器需要按系统提示放行后首次启动。
