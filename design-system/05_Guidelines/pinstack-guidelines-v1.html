PinStack 图标规范页与 UI 基线

首轮先锁定同一套图标网格、笔触、圆角、状态规则，再进入代码落地。8 个核心母版图标必须先在同页统一出现， 所有高频页面和入口都基于这套浅色主系统映射。

浅色主系统精修基线
Icon Spec

8 个核心母版图标

统一 16px 主栅格，线性主风格，1.45px 笔触，小圆角转折，默认中性色，品牌紫只做激活点缀。

截图

用边框与画幅暗示“捕获”，不使用写实相机轮廓。

录屏

核心是“记录正在发生”，圆形记录核外加轻量容器框。

复制

双层文档关系更适合工具型桌面语境，避免厚重纸片感。

编辑

斜向笔触更轻，避免传统粗笔或写实铅笔带来的陈旧感。

分类

保持文件夹语义但更扁平、更克制，适合系统层分类入口。

收藏

默认中性描边，激活时才允许小面积强调，不用金色大面积占位。

置顶

沿用产品语义最强的 Pin 形态，保留桌面工作台识别度。

设置

齿轮只做轻量结构，不走复杂工业机械风。

UI Tokens

颜色 / 圆角 / 字号 / 阴影

类别 值 用途
背景 #F5F5F3 应用背景 / 主基底
面板 #FBFBFA 主面板 / 工作台内容层
品牌 #7C5CFA 主 CTA / 选中态 / focus / 锚点
圆角 8 / 12 / 16 / 20+ 按钮输入 / 卡片 / 面板 / 主容器
字号 12 / 13 / 14 / 16 / 20 / 24 / 32 正文默认 14，页面标题 24，欢迎页大标题 32
阴影 极轻 / 轻 / 中 只服务层级，不制造“炫感”
States

默认 / Hover / Active / Disabled

状态 规则
Default 中性色描边，留白优先，图标与文字对比度适中。
Hover 仅微提升背景与前景对比，允许极轻浮起，不出现厚阴影。
Active 色值轻微加深，品牌紫仅用于主动作或明确选中。
Disabled 降低对比与不透明度，保留结构，不做灰雾模糊。
Component Rules

共享组件唯一基线

  • Button / Input 统一 8px 圆角，默认高度 36-40px。
  • Card 统一 12px 圆角，内容优先，工具按钮弱存在感。
  • Panel / Drawer / Settings / Help 统一 16px 圆角，靠留白分组，不靠粗线。
  • Sidebar 像工作台导航，不像企业后台；选中态使用品牌软底，不用重色块。
  • Search 是 Dashboard 唯一主焦点；一个区域最多一个主按钮。
