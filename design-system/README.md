# PinStack Design System v1

本目录用于承接首轮“母版优先”交付物，先锁定规范和高频资产，再映射到代码。

## 目录

- `01_Brand_Icons`
  - App / Dock / Menubar / 桌面悬浮入口的品牌层资产说明
- `02_System_Icons`
  - 核心系统图标母版与扩展图标清单
- `03_UI_Masterboards`
  - Dashboard / Capture Hub / Settings / Help / Onboarding / 悬浮入口展开态的高保真静态统一稿
- `04_Stateboards`
  - 空状态、展开收起、悬浮弱化、卡片 hover/selected 等关键状态稿
- `05_Guidelines`
  - 图标规范页、token 摘要页、组件统一规则页

## 本轮范围说明

- 仅精修浅色主系统。
- 所有新增内容与高频入口必须优先切到新系统。
- 低频遗留图标和长尾状态图本轮不强推替换，保留到下一轮扩展。
